# Details
* OS        : Arch Linux     
* WM        : bspwm    
* Terminal  : kitty    
* Shell     : zsh          
* Bar       : polybar     
* Lockscreen: not yet     
* Font Used : JetBrains Mono
* Compositor : [picom (Pijulius Fork)](https://github.com/pijulius/picom)
